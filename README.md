# ChartJS Plugin for Wordpress</br>

<img src="https://i.ibb.co/jwx3TV2/chart.png" />

## Installation

1. Unzip file in your /wp-content/plugins directory.
2. Activate plugin in wp-admin.
3. Create chart in admin and show it on front-end
